# CLI trivia game
A simple CLI trivia game built using Javascript(Node.js). User answers questions and gets feedback on the results.

## Features
An interactive CLI menu
A timer for the quiz
Results feedback

## Technologies Used
Javascript
chalk
@inquirer/prompts

## How to Run
1. Run index.js or trivia 
2. Select Start game from the menu
3. Answer each question  and press enter
4. Results will be provided at the end
